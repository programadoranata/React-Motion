# React-Motion

# [cite_start]Atividade Prática: Animações com Framer Motion [cite: 1]

[cite_start]Este projeto demonstra o uso da biblioteca Framer Motion para transformar interfaces estáticas em dinâmicas. [cite: 3]

## ✅ Requisitos Implementados

* [cite_start]**Parte 1:** Card com animação de entrada (1.2s) usando `initial`, `animate` e `transition`. [cite: 11, 14, 17]
* [cite_start]**Parte 2:** Botão com interações de `whileHover` e `whileTap`. [cite: 22, 28]
* [cite_start]**Parte 3:** Lista em cascata utilizando `Variants` e `staggerChildren`. [cite: 31, 37, 38]
* [cite_start]**Parte 4:** Alerta com animação de saída usando `AnimatePresence` e `exit`. [cite: 43, 47]
