# React-Motion

# Atividade Prática: Animações com Framer Motion [cite: 1]

Este projeto demonstra o uso da biblioteca Framer Motion para transformar interfaces estáticas em dinâmicas. [cite: 3]

## ✅ Requisitos Implementados

**Parte 1:** Card com animação de entrada (1.2s) usando `initial`, `animate` e `transition`.
**Parte 2:** Botão com interações de `whileHover` e `whileTap`.
**Parte 3:** Lista em cascata utilizando `Variants` e `staggerChildren`.
**Parte 4:** Alerta com animação de saída usando `AnimatePresence` e `exit`.
